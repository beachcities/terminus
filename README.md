# Terminus

A concept note on the boundary of safe delegation in the age of AI agents — where machine-readable and machine-understandable end, and "may the machine *use* it?" begins.

*Terminus* — the Roman boundary stone that publicly marked what belonged to whom and how far. A companion to [Sphragis](https://github.com/beachcities/sphragis), the seal that decides whether an operation may proceed.

# Terminus（日本語）

AIエージェント時代における「安全な権限委譲の境界線」についての構想ノート。機械可読・機械理解可能性が終わり、「機械がそれを*扱ってよいか*」が始まる地点を扱う。

*Terminus*（テルミヌス）は、何が誰のものでどこまでかを公示したローマの境界石。操作を許してよいかを判定する印章 [Sphragis](https://github.com/beachcities/sphragis) と対をなす。
